# Trade War Lab Simulation Web App

This is a web-based simulation for U.S.-China trade negotiations. Instructors create and manage classes, invite students, assign teams, advance the simulation through each phase, and review negotiation activity. Students join classes, receive team-specific briefings, negotiate in real time, chat in team/global channels, and vote on trade proposals

## Current Scope 

- Instructor authentication and dashboard
- Student authentication and dashboard
- Invite-based or code-based class enrollment
- Automatic class setup with USA and China teams
- Automatic briefing seeding for each team and interest group
- Trade item seeding and editing
- Multi-phase simulation flow:
  - `0`: Setup
  - `1`: Domestic Negotiation
  - `2`: Bilateral Negotiation
  - `3`: End
- Real-time class/team chat via Supabase Realtime
- Proposal creation, voting, execution, and team score updates
- Optional NotebookLM link per class for research support
